# YouTube cookies 导出教程
#
# 由于 YouTube 反爬限制，需要浏览器 cookies 才能下载。
#
# 方法 1 - 使用 Get cookies.txt LOCALLY 插件（推荐）:
# 1. Chrome 安装 Get cookies.txt LOCALLY 扩展
# 2. 访问 https://www.youtube.com 并登录你的 Google 账号
# 3. 点击扩展图标 → Export 导出
# 4. 将导出的 cookies.txt 文件放到本项目根目录
# 5. 重启服务即可
#
# 方法 2 - 手动提取:
# Chrome: F12 → Application → Cookies → youtube.com
# 导出 Netscape 格式 cookie 保存为 cookies.txt
